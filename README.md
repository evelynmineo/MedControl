MedControl

Descrição

O MedControl é um aplicativo mobile desenvolvido em React Native com Expo para auxiliar usuários no controle de medicamentos. O sistema permite cadastrar medicamentos, visualizar a lista de remédios cadastrados e acompanhar o histórico de medicamentos administrados.

Funcionalidades

* Tela de Login
* Menu Principal
* Cadastro de Medicamentos
* Visualização de Medicamentos
* Histórico de Medicamentos
* Marcar medicamento como tomado

Tecnologias Utilizadas

* React Native
* Expo
* JavaScript
* React Navigation

Como Instalar e Executar

Pré-requisitos

* Node.js
* NPM
* Expo CLI

Instalação

1. Clone o repositório:

git clone https://github.com/evelynmineo/MedControl

2. Entre na pasta do projeto:

cd MedControl

3. Instale as dependências:

npm install

4. Execute o projeto:

npx expo start

5. Abra o aplicativo Expo Go e escaneie o QR Code exibido no terminal.
